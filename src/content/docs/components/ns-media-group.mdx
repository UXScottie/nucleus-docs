## Guidance

:::do
- Use media that are connected together to show the process of the story.
- Place the media in the correct order.
- Add appropriate `ratio` for each of the media.
- Use `alt` to narrate the story textually. 
:::

:::dont
- Use `bento` type for other purpose than to narrate a story.
:::

#### Specification notes

##### `media` attribute

The structure of the each of `media` object.

| Key | Options | Description |
| :--- | :--- |:--- |
|`type`| `image` | The type of the media that needs to be displayed. |
|`src`| Refer [Assets](/guidelines/assets#placeholder-images) for choice of image sizes. | The link to the media. |
|`ratio`| `1 / 1` `4 / 3` `4 / 5` `16 / 9` | The aspect ratio of the media. |

```json
[
  {
    "type": "image",
    "src": "/placeholder-4x5-360x450px.webp",
    "ratio": "4 / 5"
  },
  {
    "type": "image",
    "src": "/placeholder-4x3-480x360px.webp",
    "ratio": "4 / 3"
  },
  ...
]
```
