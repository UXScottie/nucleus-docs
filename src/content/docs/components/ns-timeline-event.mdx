## Guidance

:::do
- Use text or `ns-card` within `additional content` to provide additional information.
- Use either default, icon or number decoration for all events in the timeline sequence for the `standard` variant.
- Use either number or fallback decoration for all events in the timeline sequence for the `Step` variant.
:::

:::dont
- Add status to an event for the `step` variant.
- Mix two different decoration types in events.
- Use any icon type with names ending `solid` or `outline`.
:::
