## Overview

<CompDetails name={componentName} />

The `ns-tabs` component separates related content into clearly labelled sections. It comprises multiple `ns-tab` components.

### Standard

![Labelled diagram of ns-tabs](@assets/components/ns-tabs/guidance.webp)

#### Key

| Key | Field type | Guidelines |
| :--- | :--- | :--- |
| <span class="marker">A</span> | Icon | This is the icon inside the tab. This should relate to the label text. [See our icons](../guidelines/assets/#icons). |
| <span class="marker">B</span> | Heading | Use 1 or 2 words to inform the user of information found within each panel below. |
| <span class="marker">C</span> | Selected Tab | This is the visual state of a selected tab. Used to show the user which content is visible below. |
| <span class="marker">D</span> | Panel | This is the area in which the content for each [`<ns-tab>`](./ns-tab) will be displayed. |
