## Guidance

:::do
- Use to show a block of content is loading.
- Use within a component slot.
- Use if content will load under 5 seconds.
- Use if part of the page is updating or loading.
:::

:::dont
- Use for partial content.
- Use for actionable elements like [`<ns-tabs>`](./ns-tabs) or [`<ns-cta>`](./ns-cta).
- Use for media.
:::
