## Overview

<CompDetails name={componentName} />

The `ns-tab` is the button that changes the content in the `ns-tabs` when clicked. It shows the user relevant content with a heading and an optional icon.

:::caution
This component can only be used within `ns-tabs`. Refer `ns-tabs` [slots specification table](/components/ns-tabs#specification) for usage.
:::

### Examples

<ExampleViewer type="standard" />

## Guidance

:::do
- Always add heading text.
:::

:::dont
- Add an icon to one tab and not to the other tabs.
- Add more than 2 words to the heading.
- Add different types of icons (solid, outline, functional).
:::
