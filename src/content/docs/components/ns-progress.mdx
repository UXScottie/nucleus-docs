### Standard

![Labelled diagram of ns-progress](@assets/components/ns-progress/guidance.webp)

#### Key

| Key | Field type | Guidelines |
| :--- | :--- | :--- |
| <span class="marker">A</span> | Step Count | This text informs you of the current step you are on and how many steps there are in total. |
| <span class="marker">B</span> | Info | This is a heading statement about this particular step. It is optional. |
| <span class="marker">C</span> | Bar | This represents the current step and those that have been already been completed. They overlay the track. |
| <span class="marker">D</span> | Track | This represents each step there is in in the process in total. Up to 20 steps are permitted. |
| <span class="marker">E</span> | Next | A statement of what the following step will be. It is optional. |
| <span class="marker">F</span> | Completed | The status shown on completion of the journey. |
